∎ 뉴스기사 크롤러

1. 기사 크롤링
 1) 선택한 신문사의 전체 기사를 날짜별로 크롤링
ex) 한국일보\2021\07 – 파일명은 한국20210701 (같은 날짜의 기사는 한 개 파일로 정리)
 2) 네이버 인링크 기사만 크롤링
 3) 크롤러 실행시 변수는 날짜로, 선택한 신문기사의 검색 결과를 특정 기간 동안(예 1년~10년) 검색하여 폴더별로 저장
   폴더: 년 - 월
 4) 저장 결과 (개별 파일)
날짜		신문사		제목(헤드라인)				본문
20210701	한국일보		산 따라 강 따라 나뉜 말의 지도		언어 지도가…
 5) 입력할 수 있는 변수 – 시간 / 신문사 편집 가능
 6) 중요! 오류가 발생했을 때
   - 멈추지 않고 스킵할 수 있게
   - 오류 기사는 url만 따로 저장 (이 파일에  실제 크롤링 기사 / 전체 대상 기사)

 - 본문 링크 추가
 - 댓글은 추후에 추가

2. 단어 검색기
 1) 특정 단어가 포함된 기사를 크롤링된 기사 중에서 추출
   - 월 단위로 한 개의 파일 생성
 2) 불필요한 정보 삭제 ex) ▶ 경향신문 최신기사, ▶기사 제보하기 등
 3) 저장 결과는 크롤링 결과와 동일(tab으로 구분)

@@ 결과물은 모두 텍스트파일로, 구분은 탭으로
     크롤링은 일 단위로 파일 생성, 검색기는 월 단위로 파일 생성



입력 변수는 시간 ex)2022-07-01
신문사 편집가능
폴더저장 tree
연 > 월 > 일별
크롤링은 생으로 크롤링 텍스트 처리는 검색기에서
날짜	분류 	신문사	제목	본문	본문링크

오류 처리
스킵후에 오류기사 url 저장
전체 기사중 스킵된 기사 개수 저장

검색기

월 단위로 한 파일 생성
